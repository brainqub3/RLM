## Analytics

### Motif Velocity
Rate of new motif generation over time
```
MV = (new_motifs / time_elapsed)
```

### Field Coherence Contribution
Measure of how entity's motifs enhance overall field stability
```
FCC = (stabilizing_motifs / total_motifs)
```

### Autonomy Index
Ratio of self-initiated to prompted actions
```
AI = (self_initiated / total_actions)
```
